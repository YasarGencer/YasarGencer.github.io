Siege Wizard

PUZZLE

Itch.io GitHub YouTube

This project was made for GGJ 2022. And it was my first game jam experience. Theme was duality.

We created a wizard who can time travel to before and after the invasion. That was our duality. This is a puzzle game which you try to reach the door in correct time. There is a unique time changing mechanic in this game and it works like a charm.

There is also a box player can grab and a box player can use the force to move. With these elements player can find its way to the door trough the levels.
